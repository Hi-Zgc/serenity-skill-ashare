# 行为准则

本项目讨论 A 股公开资料研究、数据源接入和 Agent Skill 设计。请把讨论集中在证据、代码和流程上。

## 基本要求

- 用事实和可复现步骤表达问题。
- 质疑结论时说明依据、来源和缺口。
- 区分“数据错误”“来源不可用”“研究判断不同”。
- 尊重不同市场经验和研究风格。
- 避免把个股讨论变成拉票、喊单或人身攻击。

## 不接受的行为

- 发布买卖号召、收益承诺或诱导交易。
- 传播未核实传闻并包装成事实。
- 借 Serenity 或其他公开账号名义组织跟单、喊单或攻击异议者。
- 公开他人隐私、账号、cookie、token 或非公开调研材料。
- 使用侮辱、骚扰、歧视性语言。
- 反复提交与项目目标无关的推广内容。

维护者可以关闭偏离项目目标或存在明显风险的讨论。
